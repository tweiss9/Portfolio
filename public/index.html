Projects

Spotify Playlist Sorter

I developed a Flask web app that sorts user playlists using HTML, CSS, and JavaScript. It interacts with the Spotify Web API to retrieve and display personalized playlist data. The app allows users to dynamically sort their playlists based on release date, track name, and artist name. Additionally, it offers the option to update existing playlists or create new ones. To enable real-time sorting without server-side processing, the playlist organization is handled using client-side JavaScript.
